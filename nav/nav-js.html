JavaScript in easy steps instructs you how to add functionality (logic) and dynamic effects (DHTML) to web pages. It contains separate chapters on all major features of the JavaScript language. There are code examples and browser screenshots illustrating each aspect of JavaScript. This popular title includes examples showing how to develop Rich Internet Applications (RIAs) using JavaScript Object Notation (JSON) and the latest techniques employing Asynchronous JavaScript And XML (AJAX).
